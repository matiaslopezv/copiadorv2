***INSTRUCCIONES***
1-Debes cargar tus scripts en un archivo .xlsx donde el contenido este distribuido en 2 columnas
    (recordar: las columnas son verticales) A1 y B1, donde A1 correspondera al titulo de tu script
     y B1 al contenido de tu script.
2-No debes dejar espacios entre cada fila. (recordar: las filas son horizontales)
3-Ya puedes cargar tus scripts personalizados para ser mostrados en el ~COPIADOR~.
4-Mediante el boton "Elegir Archivo" debes seleccionar tu .xlsx donde guardaste tus scripts
    segun los pasos 1 a 3.
5-Una vez cargados ya puedes usar la barra de busqueda y filtrar segun tus necesidades.
6-Puedes usar el boton "Limpiar" para borrar el contenido de la barra de busqueda y volver
    a mostrar nuevamente el contenido completo.
7-Haz click izquierdo en cualquier script que desees y este se copiara automaticamente a tu 
    portapapeles.
